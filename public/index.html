Content Delivery Networks (CDNs) are distributed server networks that cache and deliver web content from locations closer to users. They're commonly used to improve website performance, reduce latency, handle high traffic loads, and provide DDoS protection.

This tool identifies CDN providers by analyzing multiple data sources. Enter a domain to perform DNS analysis, ASN lookup, and reverse DNS resolution to determine the CDN infrastructure.

Detection Methods

  • DNS Analysis: Examines DNS records and CNAME chains for CDN-specific domains
  • ASN Lookup: Analyzes Autonomous System Numbers to identify CDN provider networks
  • HTTP Headers: Checks response headers for CDN-specific signatures
  • Reverse DNS: Performs reverse DNS lookups to map CDN infrastructure

Currently Supported CDN Providers

  • Akamai
  • Cloudflare
  • Fastly
  • AWS CloudFront
  • Google Cloud CDN
  • Microsoft Azure CDN
  • Alibaba Cloud CDN

Confidence scores are calculated based on the strength and quantity of evidence found. Each detection method contributes different weights to the final score, and all evidence used in the calculation is displayed in the results.
